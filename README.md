# Blinkit Sales Dashboard

## 📌 Overview
Analyzed 10,000+ sales records to identify trends and built
an interactive Power BI dashboard for business insights.

## 🛠️ Tools Used
- Power BI
- Excel
- SQL (MySQL)

## 📊 Key Findings
- Built 5+ KPIs including revenue and category performance
- Identified top 3 product categories = 60% of total revenue
- Improved reporting accuracy by cleaning & transforming raw data

## 📁 Files
- `blinkit_data.xlsx` — Raw and cleaned dataset
- `blinkit_dashboard.pbix` — Power BI dashboard file
- `blinkit_queries.sql` — SQL queries used for analysis
